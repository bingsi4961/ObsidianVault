---
title: 常見的 Content-Type 值
tags: [程式開發通識]

---

```htmlmixed=
Response.ContentType = "application/json";                    // JSON 格式
Response.ContentType = "application/json; charset=utf-8";     // JSON 格式（指定編碼）
Response.ContentType = "text/plain";                         // 純文字
Response.ContentType = "text/html";                          // HTML
Response.ContentType = "application/xml";                    // XML
```
